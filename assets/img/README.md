# Profile photo

Drop your headshot here as **`profile.jpg`** (this exact path: `assets/img/profile.jpg`).

- The hero avatar picks it up automatically — no code changes needed.
- Until a `profile.jpg` exists, the hero shows a "GSK" monogram fallback.

## Recommended
- Square image (e.g. 600×600 or larger), face centered.
- JPG or PNG. If you use PNG, either rename it to `profile.jpg` or update the
  `src` in `index.html` (`.hero-avatar img`).
- Keep it reasonably small (< ~300 KB) for fast loading.
