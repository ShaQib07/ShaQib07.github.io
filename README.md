# Golam Shakib Khan — Portfolio

Personal developer portfolio, built with plain HTML, CSS and JavaScript (no build step) and deployed via GitHub Pages.

Live at: https://ShaQib07.github.io

## Structure

```
index.html            Page markup (Hero, About, Skills, Experience, Projects, Achievements, Contact)
assets/css/style.css   Dark theme styling, layout, responsive rules, animations
assets/js/script.js    Scroll reveal, sticky nav, mobile menu toggle
```

## Local preview

Any static file server works, e.g.:

```
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deployment

This repo is named `ShaQib07.github.io`, so GitHub Pages serves it automatically from the root of the
default branch. See repository Settings → Pages to confirm the source is set to the `main` branch, `/ (root)`.
